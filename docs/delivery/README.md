# 比赛交付材料目录

本目录用于沉淀比赛答辩、演示视频、截图、真实数据准备和最终提交包相关材料。

## 文件列表

| 文件 | 用途 |
|---|---|
| `ppt_outline.md` | PPT 答辩大纲、每页标题、讲述口径和配图建议 |
| `project_book.md` | 项目书底稿，可用于申报书、项目说明书或答辩附件 |
| `defense_script.md` | 答辩讲稿和问答口径 |
| `demo_video_script.md` | 演示视频分镜脚本 |
| `screenshot_shotlist.md` | 截图素材清单 |
| `real_data_worklist.md` | 真实 GIS/视觉数据准备工作清单 |
| `submission_package_template.md` | 最终提交包目录模板和检查项 |

## 使用顺序

1. 先用 `project_book.md` 统一项目定位、建设内容和技术路线。
2. 用 `ppt_outline.md` 搭出 PPT，并按 `screenshot_shotlist.md` 补图。
3. 用 `demo_video_script.md` 录制演示视频。
4. 用 `defense_script.md` 准备答辩。
5. 最终按 `submission_package_template.md` 打包提交。
