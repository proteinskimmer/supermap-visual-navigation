## 4. 截图规范

- 分辨率建议 1920x1080。
- 浏览器缩放 100%。
- 使用浅色/清晰主题。
- 截图前关闭控制台、下载栏、通知弹窗。
- 文件命名建议：

```text
screenshots/S-09_candidate_routes.png
screenshots/S-15_replanning_compare.png
```
