# 分工细化：航线规划、风险校验与动态重规划

## 1. 角色目标

实现基于 GIS 约束的自动航线规划、风险评分和动态重规划，支撑系统核心演示。

## 2. 输入数据

| 输入 | 来源 | 说明 |
|---|---|---|
| 起点/终点/任务点 | 前端/任务配置 | `[lon, lat, height]` |
| 任务区域边界 | GIS | 限制规划范围 |
| DEM 高程 | GIS | 计算安全高度和高程剖面 |
| 风险区/禁入区 | GIS/前端 | 用于避让和评分 |
| 建筑/障碍物 | GIS | 用于安全距离校验 |
| 参数 | 前端 | 最低高度、最高高度、最大航程、权重 |

## 3. 输出数据

| 输出 | 说明 |
|---|---|
| 候选航线 | 最短、最安全、综合最优 |
| 风险评分 | 0 到 100 分 |
| 风险等级 | low/medium/high/critical |
| 风险航段 | 每个风险段的起止点和原因 |
| 高程剖面 | 距离、高程、飞行高度 |
| 重规划结果 | 新航线、旧航线风险段、触发原因 |

## 4. 算法优先级

### P0：可演示版本

- 使用二维栅格 A*。
- 风险区设置为高代价或不可通行。
- 生成三种模式：
  - 最短：距离权重最大。
  - 最安全：风险代价最大。
  - 综合：距离、风险、平滑度综合。
- 风险校验支持多边形相交、缓冲区距离、最大航程。

### P1：增强版本

- 加入 DEM 高程约束。
- 输出高程剖面。
- 增加转弯角惩罚，减少锯齿路径。
- 支持局部重规划。

### P2：扩展版本

- 增加 RRT* 或采样规划。
- 支持更多动态障碍。
- 支持离线数据包规划。

## 5. 成本函数建议

```text
cost = distance_weight * distance
     + risk_weight * risk_cost
     + terrain_weight * terrain_cost
     + smooth_weight * turn_penalty
```

三种模式建议权重：

| 模式 | distance | risk | terrain | smooth |
|---|---:|---:|---:|---:|
| shortest | 1.0 | 0.3 | 0.2 | 0.1 |
| safest | 0.5 | 2.0 | 0.6 | 0.2 |
| balanced | 1.0 | 1.0 | 0.5 | 0.3 |

## 6. 风险评分建议

初始分数 100，根据风险扣分：

| 风险项 | 扣分建议 |
|---|---:|
| 穿越禁入区 | 直接 critical |
| 穿越高风险区 | -30 |
| 靠近障碍物缓冲区 | -10 到 -25 |
| 超出任务边界 | 直接 critical |
| 低于安全高度 | -20 |
| 航程超限 | -20 |
| 高程突变明显 | -5 到 -15 |

等级：

| 分数 | 等级 |
|---|---|
| 85 到 100 | low |
| 65 到 84 | medium |
| 40 到 64 | high |
| 0 到 39 | critical |

## 7. 执行步骤

### 第 1 步：简化栅格规划

- 将任务区域离散成栅格。
- 标记风险区、禁入区、障碍物。
- 实现 A*。
- 输出 GeoJSON LineString 或路线点数组。

### 第 2 步：多模式规划

- 调整代价权重。
- 一次请求返回三条候选航线。
- 每条航线附带长度、预计时间、风险分。

### 第 3 步：风险校验

- 对航线分段。
- 判断与风险区是否相交。
- 判断距离障碍物是否小于安全缓冲。
- 输出风险段和原因。

### 第 4 步：高程剖面

- 按路线采样。
- 获取或模拟地形高程。
- 输出距离-地形高程-飞行高度数组。

### 第 5 步：动态重规划

- 在仿真过程中加入临时风险区。
- 判断当前航线剩余段是否受影响。
- 以当前位置为新起点重新规划到目标点。
- 输出旧风险段和新航线。

## 8. 验收标准

- 能基于同一任务生成三条不同航线。
- 至少一条航线能明显避开风险区。
- 风险原因能被前端直接展示。
- 动态风险区加入后能触发重规划。
- 所有结果坐标能准确叠加到 SuperMap 三维场景。
