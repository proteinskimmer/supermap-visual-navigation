# SuperMap 接入验收清单

## 1. iDesktopX 验收

- [ ] iDesktopX 2025 可启动。
- [ ] 工作空间可打开。
- [ ] 任务区域边界已整理。
- [ ] 影像、DEM、矢量图层可正确叠加。
- [ ] 坐标系统一且已记录。
- [ ] 风险区图层 `risk_zone` 已建立。
- [ ] 障碍物图层 `obstacle` 已建立。
- [ ] 三维场景已制作。
- [ ] 默认视角已设置。
- [ ] 工作空间路径已记录。

## 2. iServer 验收

- [ ] iServer 2025 可启动。
- [ ] 管理页面可登录。
- [ ] 三维服务发布成功。
- [ ] 地图服务发布成功。
- [ ] 数据服务发布成功。
- [ ] 浏览器可访问服务 URL。
- [ ] 服务地址已记录到 `04_service_url_registry_template.md`。
- [ ] 服务地址已写入 `config/supermap_services.local.json`。

## 3. 前端接入验收

- [ ] `SuperMapScene.vue` 能进入 SuperMap 接入模式。
- [ ] 页面能初始化 Viewer。
- [ ] 页面能加载三维服务。
- [ ] 默认视角定位到任务区域。
- [ ] 风险区叠加位置正确。
- [ ] 航线叠加位置正确。
- [ ] 动态重规划航线叠加正确。
- [ ] 视觉候选区域叠加正确。
- [ ] mock 备用视图仍可使用。

## 4. 后端接入验收

- [ ] 后端能读取服务配置。
- [ ] 后端能知道 iServer 数据服务地址。
- [ ] 风险区/障碍物真实数据接入方案明确。
- [ ] 若暂不接真实数据，已明确继续使用 demo JSON。

## 5. 演示验收

- [ ] 前端打开后可见 SuperMap 三维场景。
- [ ] 点击规划能看到三条候选航线。
- [ ] 风险区和航线空间位置一致。
- [ ] 点击重规划能看到新旧航线对比。
- [ ] 视觉候选区域能叠加展示。
- [ ] 有至少 3 张截图可用于 PPT。
