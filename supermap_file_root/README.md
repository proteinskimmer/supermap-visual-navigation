# SuperMap file root

This directory is the project-side iServer file root.

Current purpose:

- `demo_workspace/`: target output for the project iDesktopX workspace.
- `upload/`: temporary files prepared for iServer import or upload.
- `published/`: notes or exported artifacts after iServer publication.

Target workspace:

```text
E:\supermap_project\supermap_file_root\demo_workspace\low_altitude_demo.smwu
```

Do not confuse this project workspace with the built-in `3D-CBD` sample service. `3D-CBD` only proves that the SuperMap runtime chain works. The competition demo should eventually use `low_altitude_demo` services published from the workspace above.
